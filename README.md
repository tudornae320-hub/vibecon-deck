# VibeCon — Pitch Deck

Static, single-file pitch/sponsor deck for **VibeCon** (Bucharest, November 2026).

- `index.html` — the live deck. Responsive, self-contained (no build step, fonts via Google Fonts CDN). Open it directly or serve the folder.
- `index-print.html` — fixed 1920×1080 canvas version, used to render the PDF.
- `VibeCon-Sponsor-Deck.pdf` — exported sponsor deck.
